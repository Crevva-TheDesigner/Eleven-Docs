# **App Name**: Crevva

## Core Features:

- Product Browsing and Search: Implement a search feature to enable users to quickly find specific products. Categories to filter notes, code files, journals and planners.
- Secure Purchase and Download: Allow users to securely purchase products and download files immediately after payment.
- AI-Driven Product Recommendations: Suggest relevant digital products to users based on their browsing history, purchase history, and product metadata. Use a tool to decide which metadata is the most relevant.
- Ratings and Reviews: Implement a system for buyers to rate and review products to ensure quality and build trust within the marketplace.
- Sales analytics dashboard: Provide visual sales metrics to vendors to enable data driven decisions.

## Style Guidelines:

- Primary color: Teal (#008080) for a modern and vibrant feel, as requested.
- Background color: Soft gray (#D3D3D3) for a neutral backdrop that complements the teal, as requested.
- Accent color: White (#FFFFFF) to provide a clean contrast and enhance readability, as requested.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern appearance that works well for both headings and body text.
- Use minimalist icons in teal or soft gray to maintain a consistent and modern theme.
- Maintain a clean, grid-based layout with ample whitespace to ensure a professional and user-friendly experience.
- Use subtle animations for product transitions and interactive elements to enhance user engagement.
